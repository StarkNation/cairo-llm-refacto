img_viz, labels_viz = next(data_iter)


Visualiser les images à l'aide de matplotlib :

Pour cela, nous devrons d'abord importer la bibliothèque matplotlib, puis définir une fonction pour visualiser les images.


import matplotlib.pyplot as plt
import numpy as np

# Fonction pour dénormaliser et visualiser une image
def imshow(img):
    img = img / 2 + 0.5  # dénormaliser
    npimg = img.numpy()
    plt.imshow(np.transpose(npimg, (1, 2, 0)))
    plt.show()

# Visualiser les premières images du lot
num_images_to_show = 4  # ou tout autre nombre de votre choix
for i in range(num_images_to_show):
    imshow(img_viz[i])
    print(f"Label: {labels_viz[i]}")

Notez que la fonction imshow dénormalise l'image pour l'afficher correctement. La ligne img = img / 2 + 0.5 est basée sur l'hypothèse que nous avons normalisé les images avec transforms.Normalize((0.5,), (0.5,)) comme mentionné précédemment.
